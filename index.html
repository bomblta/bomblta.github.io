Минимальные требования для запуска сервиса

System
MacOS 10.11.6 и выше, Ubuntu 18 и выше, Debian 10.3 и выше
при условии поддержки блиотеки libzmq4
Language
Go (Golang)
компилируемый многопоточный язык программирования
Library
Предустановлена библиотека libzmq4
Ядро
Предустановленно и сконфигурировано ядро Биткоина (Bitcoin Core) версии 0.19.1
RAM
Оперативная память объемом 8 Гб
(рекомендовано 16ГБ)
Processor
Двухъядерный 2,5 GHz Intel Core i5
рекомендовано четырехъядерный 3,2 GHz Intel Core i7
Storage
100+ Гб для сервиса Wallets и хранения истории , 1 Тб для хранения данных блокчейна
